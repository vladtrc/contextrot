# AGENTS Instructions

## Markdown Editing Rules

When editing any `.md` content file (posts, pages, notes), follow these rules:

0. Content types are different:
   - `_posts/` = published posts. Treat as stable content and avoid rewrites unless explicitly requested.
   - `_drafts/` = working drafts. New ideas, experiments, and structural rewrites are allowed when requested.
1. Before making edits, read other existing articles from `_posts/` to calibrate style, tone, rhythm, vocabulary, and narrative voice.
3. Do not "polish away" the original vibe.
4. Do not change meaning unless the user explicitly asks for a rewrite.
5. In research-style drafts, include a thought-provoking, counterintuitive idea that applies beyond one technology.
